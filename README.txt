DESCRIPTION
codePy.py est une application en ligne de commande qui permet de
  - determiner si un enemble de mot à partir d'un alphabet est un code ou pas
  - dans le cas écheant on affiche les caractérisques :
  		. l'entropie
  		. la longueur moyenne
  		. la longueur minimale
  		. le rendement
  		. la redondance
  		. la linéarité
  		etc.
UTILISATION
pour utiliser ce programme il faut installer python 3.x
pour exécuter utiliser tout simplement "python3 codePy.py"
